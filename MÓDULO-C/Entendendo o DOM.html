ENTENDENDO O DOM


O QUE É?

é um modelo de objetos para documentos, presente dentro de um navegador que dá acesso aos componentes internos de um website.
O DOM NÃO RODA NO NODE.JS, PRESENTE QUANDO ESTÁ RODANDO JAVASCRIPT NO NAVEGADOR

ARVORE DOM

window -> significa janela e basicamente a janela do proprio navegador ja é um objeto DOM. Dentro do window, existem varios outros objetos, como location que identifica a localização do site, URL e afins. O document, amplamente usado durante o curso, que é o documento atual e o history, que registra a informação de onde se navegou e pra onde irá se navegar.
Dentro do document existe o objeto html(no caso, document é PARENT de html) e dentro de html existe o head e body (no caso, html é parent de head e body)

SELEÇÃO DE ELEMENTOS


O QUE SÃO ELEMENTOS?

são tudo e qualquer coisa que pertencente a ARVORE DOM e existem vários tipos de seletores de elementos, mas serão abordados apenas 5.

POR MARCA

getElementsByTagName() consegue selecionar multiplos objetos de um mesmo tipo/tag

POR ID

getElementsById() usado em sites mais complexos, identifica-se um div etc

POR NOME

getElementsByName() se tivermos uma propriedade name

POR CLASSE

getElementsByClassName()

POR SELETOR

querySelector() querySelectorAll() a mais nova dessas 5 opções, atualmente recomendada
