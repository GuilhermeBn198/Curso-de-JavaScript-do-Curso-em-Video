Os Operadores do Javascript

-possuem varias familias de Operadores, porém serão abordados apenas os operadores:
Aritméticos
os de Atribuição
os Relacionais
Lógicos
e os Ternários

obs: o comando typeof() é um tipo de operador também


Aritméticos

são aqueles que abordam os calculos básicos comums, como adição, subtração, multiplicação, divisão, exponenciação e divisão inteira.

ex: +, -, *, /, %, **

ex2: 5 + 2 = 7
5 - 2 = 3
5 * 2 = 10
5 / 2 = 2,5
5 % 2 = 1
5 ** 2 = 25

EXISTEM PRECEDENCIA DE OPERADORES

seguindo a ordem da matemática padrão, primeiro fazemos os () depois os números/operações, seguindo a ordem de precedencia de potencia/exponenciação entao fazemos divisão/multiplicação para então fazer adição/subtração.

Atribuição

são aqueles que atribuem valores a certos tipos de variaveis e condições

ATRIBUIÇÃO SIMPLES

"=" significa "RECEBE" ao invés de "IGUAL" ex:
var a = 5 + 3
var b = a % 5
var c = 5 * b ** 2
var d = 10 - a / 2
var e = 6 * 2 / d
var f = b % e + 4 / e

AUTO-ATRIBUIÇÃO

ex: n = 3
n = n + 4
n = n - 5
n = n * 4
n = n / 2
n = n ** 2
n = n % 5
pra simplificar as AUTO-ATRIBUIÇÃO:
ex: n = 3
n += 4
n -= 5
n *= 4
n /= 2
n **= 2
n %= 5

INCREMENTO

acrescentará ou diminuirá 1 no valor
ex: n = 10
n++ n=11
n-- n=9
