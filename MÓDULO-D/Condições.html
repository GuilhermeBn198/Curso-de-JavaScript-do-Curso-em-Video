CONDIÇÕES


sequências

sequencias são linhas de código que requerem ser de uma ordem específica no caso, de cima para baixo, geralmente todo código de programação é assim, porém as condições permitem que certas linhas de código sejam executadas antes de outras.

condições

condições são linhas de códigos especiais que podem ditar quando uma parte do código pode ser executada, como filtros ou placas de trânsito.

ex de condicionamento:

if(condição){
TRUE
} else {
FALSE
}

que é uma condição simples

e também existe o:
if(condição){
TRUE
}

que é uma ccondição composta
existe as condições aninhadas

if (condição){
} else {
if(condição2){
} else {
}
}
}

caso a condição do primeiro IF seja verdadeira, o primeiro bloco de código é executado, se for falsa executará o bloco do ELSE1 que verificará se o IF2 será verdadeiro ou falso.

existe também a condição multipla, que trabalha com várias possibilidades, além da possibilidade de TRUE e FALSE, com condições feitas manualmente. MUITO util em situações expecíficas.
switch(expressão) { case valor1: break case valor2: break case valor3: break case valor4: break default: break } funciona quando o comando testa a expressão inicial, e verifica o valor da expressão e testa em cada um dos CASES até achar um que satisfaz a condição do case.
