Os Operadores do Javascript

-possuem varias familias de Operadores, porém serão abordados apenas os operadores:
Aritméticos
os de Atribuição
os Relacionais
Lógicos
e os Ternários

obs: o comando typeof() é um tipo de operador também


ARITMÉTICOS

são aqueles que abordam os calculos básicos comums, como adição, subtração, multiplicação, divisão, exponenciação e divisão inteira.

ex: +, -, *, /, %, **

ex2: 5 + 2 = 7
5 - 2 = 3
5 * 2 = 10
5 / 2 = 2,5
5 % 2 = 1
5 ** 2 = 25
5 == 2 = FALSE

EXISTEM PRECEDENCIA DE OPERADORES

seguindo a ordem da matemática padrão, primeiro fazemos os () depois os números/operações, seguindo a ordem de precedencia de potencia/exponenciação entao fazemos divisão/multiplicação para então fazer adição/subtração.
ABAIXO DOS ARITMÉTICOS, ESTÃO OS RELACIONAIS. DA ESQUEDA PRA DIREITA.
por fim a ordem de precedencia dos lógicos é ! depois && e depois ||

ATRIBUIÇÃO

são aqueles que atribuem valores a certos tipos de variaveis e condições

ATRIBUIÇÃO SIMPLES

"=" significa "RECEBE" ao invés de "IGUAL" ex:
var a = 5 + 3
var b = a % 5
var c = 5 * b ** 2
var d = 10 - a / 2
var e = 6 * 2 / d
var f = b % e + 4 / e

AUTO-ATRIBUIÇÃO

ex: n = 3
n = n + 4
n = n - 5
n = n * 4
n = n / 2
n = n ** 2
n = n % 5
pra simplificar as AUTO-ATRIBUIÇÃO:
ex: n = 3
n += 4
n -= 5
n *= 4
n /= 2
n **= 2
n %= 5

INCREMENTO

acrescentará ou diminuirá 1 no valor
ex: n = 10
n++ n=11
n-- n=9

RELACIONAIS

são os operadores que aplicam as mesmas regras de relações de quase todas as outras linguagens de programação. com seus resultados de operação igual a booleano, TRUE ou FALSE
ex:
5 > 2 -> TRUE
7 < 4 -> FALSE
8 >= 8 -> TRUE
9 <= 7 -> FALSE
5 == 5 -> TRUE
4 != 5 -> FALSE
o sinal de igualdade restrita funciona testando se os dois valores são exatamente identicos, até mesmo os tipos dos valores
ex: 5 === 5 -> TRUE
5 === '5' -> FALSE

LÓGICOS

são três:
! -> negação (TRADUÇÃO: NÃO)
&& -> conjunção (TRADUÇÃO:E)
|| -> disjunção (TRADUÇÃO:OU)

NEGAÇÃO

depois da exclamação apenas use TRUE ou FALSE.
ex: var a = TRUE
!a = FALSE
!!a = TRUE

CONJUNÇÃO

com 4 possibilidades de resultado, a conjunção só aponta TRUE qunado os resultados analisados são semelhantes.
ex:
TRUE && TRUE -> TRUE
TRUE && FALSE -> FALSE
FALSE && TRUE -> FALSE
FALSE && FALSE -> TRUE

DISJUNÇÃO

também com 4 possibilidades de resultado, a disjunção só aponta TRUE quando os resultados são diferentes.
ex:
TRUE && TRUE -> FALSE
TRUE && FALSE -> TRUE
FALSE && TRUE -> TRUE
FALSE && FALSE -> FALSE

TERNÁRIOS

é composto por 2 simbolos, o ? e o : dentro de uma mesma expressão. agindo semelhantemente ao IF-ELSE-IF
é composto por 3 seções. ex:

teste ? TRUE : FALSE
o primeiro representa o teste lógico que dá TRUE ou FALSE.
o do meio representa o que irá acontecer caso o teste dê TRUE.
e o ultimo representa o que acontecerar caso o teste dê FALSE
ex:
var media=5.5
media>7 ? 'aprovado' : 'reprovado'

ex2:
var x = 8
var res = x%2==0 ? 5 : 9
o resto da divisão de x por 2 eh igual a 0 e como 0==0 então o operador apresentou TRUE, então a variavel res receberá 5.
