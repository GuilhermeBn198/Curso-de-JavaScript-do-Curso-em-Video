tratamentos de dados

os 3 tipos principais de dados são:
-number
-string
-boolean

podemos fazer concatenação de dados com o simbolo "+"
o javascript pode se confundir com o significado do "+" pois ele representa tanto a adição quanto a concatenação, sendo assim:
1(NUMBER) + 1(NUMBER) = 2(NUMBER)
1(STRING) + 1(STRING) = 11(STRING)

é necessário haver a conversão de tipos de dados caso vc queira fazer uma adição, pois o comando window.prompt apenas entrega o valor em STRINGS

para realizar conversão de dados

usamos Number(n) para converter os dados da variável em INTEIRO ou FLOAT
usamos String(n) para converter os dados da variável em STRING
